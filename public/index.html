All Interfaces Events MIME Types References Acknowledgments About this specification This specification is like no other — it has been processed with you, the humble web developer, in mind. The focus of this specification is readability and ease of access. Unlike the full HTML Standard, this "developer's edition" removes information that only browser vendors need know. It is automatically produced from the full specification by our build tooling, and thus always in sync with the latest developments in HTML. To read about its conception, construction, and future, read the original press release, and the blog post about its relaunch. Finally, feel free to contribute on GitHub to make this edition better for everyone!
